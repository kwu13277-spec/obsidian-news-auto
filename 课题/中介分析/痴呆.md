# 《Plasma proteomics mediate the association between degenerative joint diseases and dementia risk》研究框架梳理

> 说明：本梳理**仅基于合并 PDF 的正文部分**整理，不纳入补充材料（Supplementary Materials）内容。按 PDF 页码看，正文到第 12 页结束，第 13 页开始进入补充材料。

## 1. 文献核心研究问题

这篇文章试图回答两个核心问题：

1. 退行性关节疾病（DJD）是否会增加痴呆发生风险？
2. 如果这种风险确实存在，那么背后的生物学通路是什么，尤其是血浆蛋白组是否在其中起到了中介作用？

作者把 DJD 具体拆成三类：

- 骨关节炎（OA）
- 椎间盘退变（IVDD）
- 退行性脊柱疾病（DSD）

痴呆结局则分为三类：

- 全因痴呆（ACD）
- 阿尔茨海默病（AD）
- 血管性痴呆（VaD）

## 2. 总体研究框架

从正文和图文摘要可以看出，文章的整体逻辑是一个“**流行病学关联发现 -> 稳健性验证 -> 蛋白组机制筛选 -> 中介机制解释**”的四层框架。

作者实际上完成了以下 7 个连续步骤：

1. 建立前瞻性队列并完成基线分组
2. 用 Cox 回归检验 DJD 与痴呆风险的关联
3. 用敏感性分析和亚组分析检验结果是否稳健、是否存在异质性
4. 用倾向评分匹配（PSM）做复核验证
5. 找出与 DJD 相关的血浆蛋白
6. 找出与 ACD 风险相关的血浆蛋白
7. 识别同时连接“DJD”和“ACD”的中介蛋白，并估计中介比例

这个框架的关键特征是：

- 前半部分回答“**有没有关联**”
- 后半部分回答“**为什么会有关联**”

## 3. 分步骤研究框架详解

## 第一步：建立研究队列并定义暴露、结局与协变量

### 研究目的

这一步的目的，是先搭建一个可以做长期随访分析的总体研究样本，并把后续分析所需的三个基础元素定义清楚：

- 谁属于 DJD 暴露组，谁属于对照组
- 痴呆结局如何判定
- 需要控制哪些混杂因素

### 研究步骤

作者使用 UK Biobank 前瞻性队列数据。初始样本为 502,357 人。随后：

- 排除失访者 1,297 人
- 排除基线时已经患有痴呆者 255 人
- 最终纳入 500,805 人

在暴露定义上：

- 只要参与者在自报病史、住院 ICD-10 编码或基层医疗 Read code 中出现相关诊断，即定义为 DJD
- 基线时有 DJD 诊断者进入暴露组
- 基线时无 DJD 诊断者进入对照组
- 随访后才发生的 DJD 在主分析中被视为时间变化暴露

在结局定义上：

- 通过 UK Biobank 的首次发生记录识别痴呆
- 结局来源包括自报病史、住院记录、基层医疗记录和死亡登记
- 主要结局为 ACD、AD 和 VaD
- 随访截止到 2023 年 8 月

在混杂因素控制上，作者纳入了四大类变量：

- 人口学因素：年龄、性别、种族
- 社会经济因素：Townsend deprivation index、教育程度
- 生活方式因素：吸烟、饮酒、体力活动
- 共病因素：高血压、糖尿病、心血管疾病

### 研究结果

最终样本分布为：

- 非 DJD 对照组：406,760 人
- DJD 组：94,045 人

基线特征显示，DJD 组与对照组存在明显差异，说明如果不做统计调整，很容易产生混杂偏倚。DJD 组总体呈现出：

- 年龄更大：59.65 岁 vs 55.81 岁
- 女性更多：59.0% vs 53.3%
- 肥胖比例更高：33.2% vs 22.3%
- 吸烟者更多：49.3% vs 43.9%
- 教育程度略低
- 高血压、糖尿病和心血管疾病更常见

这一步的结果为全文后续分析奠定了基础，也解释了为什么作者必须使用多变量调整、敏感性分析和 PSM 来控制偏倚。

## 第二步：检验 DJD 是否增加痴呆风险

### 研究目的

这一步是全文最核心的因果线索起点，目的是回答：

- DJD 是否与未来痴呆风险升高相关？
- 这种关系是对所有痴呆类型都成立，还是只集中在某些亚型？
- 不同 DJD 亚型是否风险水平不同？

### 研究步骤

作者使用多变量 Cox 比例风险模型评估 DJD 与痴呆发病风险之间的关系，并采用逐层调整策略：

- Model 1：调整年龄、性别、种族
- Model 2：进一步调整教育、贫困指数、吸烟、饮酒、体力活动
- Model 3：进一步调整高血压、糖尿病、心血管疾病

主要观察的结局包括：

- ACD 风险
- AD 风险
- VaD 风险

同时，作者还分别分析了 OA、IVDD、DSD 三类 DJD 亚型。

### 研究结果

中位随访时间为 14.45 年。随访期间共发生：

- ACD 9,884 例
- AD 4,404 例
- VaD 2,224 例

在完全调整后的模型中，DJD 与痴呆风险的关系表现出明显分层：

1. DJD 显著增加全因痴呆风险  
DJD 组 ACD 风险升高，HR = 1.271，95% CI: 1.173-1.376。

2. DJD 显著增加血管性痴呆风险  
DJD 组 VaD 风险升高更明显，HR = 1.587，95% CI: 1.323-1.903。

3. DJD 与阿尔茨海默病风险关联不显著  
AD 的 HR = 1.107，95% CI: 0.982-1.248，未达到统计学显著。

不同 DJD 亚型在 ACD 上都表现出风险升高：

- OA：HR = 1.177
- IVDD：HR = 1.488
- DSD：HR = 1.569

在 VaD 上，风险更高：

- OA：HR = 1.358
- IVDD：HR = 2.156
- DSD：HR = 1.838

### 这一步说明了什么

这一步给出的核心结论是：

- DJD 不是单纯的局部骨关节问题，而可能与长期神经认知结局有关
- 这种关联主要体现在全因痴呆和血管性痴呆，而不是典型阿尔茨海默病
- 因此，文章后续机制分析自然把重点放在“炎症”和“血管功能障碍”上，而不是经典淀粉样蛋白通路

## 第三步：用敏感性分析和亚组分析检验结果是否稳健

### 研究目的

观察到 DJD 与痴呆风险有关之后，作者需要进一步证明这不是统计假象。也就是说，这一步要回答：

- 结果会不会是反向因果造成的？
- 会不会是手术、死亡竞争风险或随访中新发 DJD 扰动出来的？
- 这种风险在不同人群中是否一样？

### 研究步骤

作者设计了 7 组稳健性检验，主要包括：

- 处理随访中新发 DJD：将其排除，或在诊断时点截尾
- 排除基线后 2 年内发生痴呆者，以降低反向因果
- 排除 65 岁前发生 ACD 的个体
- 使用 Fine-Gray 模型处理死亡竞争风险
- 在完全调整模型基础上进一步控制关节或脊柱手术
- 做亚组分析，考察年龄、性别、种族、生活方式、共病、手术等是否有交互作用
- 对疼痛表型进行分层分析

### 研究结果

这一部分的结果总体上非常一致，说明主结论稳健：

1. 无论对随访中新发 DJD 做排除还是截尾处理，DJD 与 ACD、VaD 风险升高的关系仍然存在。

2. 排除早发痴呆或短期内发生痴呆的参与者后，DJD 与 ACD 风险升高的关系依然显著。

3. 处理死亡竞争风险后，结论仍未改变。

4. 进一步调整关节或脊柱手术后，DJD 与 ACD 的关联仍然存在。

5. 疼痛程度似乎会放大风险。正文指出，疼痛性 DJD 的 ACD 风险更高：
- 疼痛性 DJD：HR = 1.597
- 非疼痛性 DJD：HR = 1.315

6. 亚组分析发现存在明显效应修饰，风险尤其突出的人群包括：
- 年龄 < 55 岁者：HR = 2.254
- 缺乏日常规律活动者：HR = 1.557
- 接受过相关手术者：HR = 2.165

### 这一步说明了什么

这一步的价值在于，它把第二步观察到的关联从“可能有关”推进到“在多种假设下都成立”。尤其重要的是：

- 风险并不是因为某个单一偏倚造成的
- 低活动量和较重疾病负担可能会强化认知风险
- DJD 与痴呆之间更像是一条受炎症负荷、功能状态和血管负担共同影响的长期路径

## 第四步：用倾向评分匹配（PSM）进行复核验证

### 研究目的

由于 DJD 组和对照组在基线差异明显，作者还需要一个更严格的“类随机对照”式验证，来确认主分析不是由组间不平衡驱动的。

### 研究步骤

作者根据全部协变量估计 DJD 暴露的倾向评分，并进行 1:1 最近邻匹配，卡钳宽度不超过 propensity score logit 标准差的 0.1。匹配后：

- 检查标准化差异（SMD）
- 以匹配对 ID 为分层变量，再做 Cox 分析

### 研究结果

匹配后各协变量的绝对 SMD 都小于 0.1，说明两组基线差异已经明显缩小。此时再次分析发现：

- DJD 与 ACD 的关联方向没有变化
- 关联强度与主分析结果接近
- OA、IVDD、DSD 的结果趋势也一致

### 这一步说明了什么

PSM 的意义在于，它从另一个统计框架再次证明：

- DJD 与痴呆风险升高的关系不是简单由年龄、共病或生活方式差异“堆出来的”
- 主分析结论具有较好的可重复性和稳健性

## 第五步：识别与 DJD 相关的血浆蛋白

### 研究目的

前面几步已经证明了 DJD 与痴呆风险有关，但还不知道“中间发生了什么”。所以这一步开始进入机制层面，目标是：

- 找出 DJD 患者体内哪些循环蛋白发生了系统性变化
- 看这些变化是否能反映炎症、代谢或血管损伤等潜在路径

### 研究步骤

作者在 UK Biobank 中提取了 50,586 名具有完整蛋白组数据的参与者，基于 Olink Explore 平台测量血浆蛋白。经过质控后，共保留 2,923 个蛋白。

然后作者使用多元线性回归分析：

- 以 DJD 暴露作为解释变量
- 以蛋白表达水平作为结果变量
- 筛选出与 DJD 显著相关的蛋白

同时还对 OA、IVDD、DSD 分别做了亚型分析，并进行功能富集分析。

### 研究结果

作者发现，DJD 会显著上调一批蛋白，也会下调一批蛋白。

正文中明确提到，DJD 显著上调的代表性蛋白包括：

- COL9A1
- GAST
- LEP
- FABP4
- CHGA

显著下调的代表性蛋白包括：

- IFGBP1
- BPIFA2
- GH1

在三类 DJD 亚型中，有 371 个蛋白呈现共同上调，代表性蛋白包括：

- GAST
- LEP
- CHGA
- FABP4

不同亚型还呈现各自更有特征性的蛋白变化：

- OA：IL-1RN、FABP1、IL-6、OXT 等上调
- IVDD：MYOM3、CSRP3、ADH4 等上调
- DSD：CHIT1、REN、LAMP3 等上调

功能富集结果表明，DJD 相关蛋白主要集中在以下通路和过程：

- 免疫反应
- 炎症反应
- 细胞黏附与迁移
- 细胞因子-受体相互作用
- 补体与凝血级联
- 凋亡相关过程

### 这一步说明了什么

这一步的核心意义是把 DJD 从“骨关节局部病变”推进成“全身蛋白组状态改变”。也就是说，DJD 患者体内确实存在可检测的系统性分子特征，而且这些特征高度指向炎症和血管相关机制，为后面的中介分析提供了候选变量。

## 第六步：识别与全因痴呆（ACD）风险相关的血浆蛋白

### 研究目的

仅仅知道某些蛋白与 DJD 有关还不够，还必须进一步证明这些蛋白本身也与痴呆发生有关。只有这样，它们才有资格成为真正的“桥梁变量”。

### 研究步骤

作者继续在蛋白组样本中使用 Cox 回归分析：

- 以每个蛋白的表达水平为暴露
- 以 ACD 发生为结局
- 筛选出显著影响 ACD 风险的蛋白

之后又进一步用限制性立方样条（RCS）分析代表性蛋白与 ACD 风险之间的剂量-反应关系。

### 研究结果

作者识别出 820 个与 ACD 风险升高显著相关的蛋白。正文中点名的代表性高风险蛋白包括：

- MENT
- GFAP
- NEFL
- PLAUR
- GDF15

而与较低 ACD 风险相关的代表性蛋白包括：

- APOE
- EGFR
- ITGAV
- CLEC3B

功能富集结果显示，这些 ACD 相关蛋白主要涉及：

- 伤口愈合
- Rap1 signaling
- 细胞因子-受体相互作用
- MAPK signaling
- 细胞外基质重塑

RCS 分析还显示，蛋白表达与 ACD 风险之间并不总是简单线性关系：

- MENT、GFAP、NEFL 呈非线性关系
- GDF15、PLAUR、FSTL1 也呈剂量-反应变化

### 这一步说明了什么

这一步确认了“蛋白组改变”和“痴呆风险”之间确实存在统计连接，因此作者可以把前一步的 DJD 相关蛋白与这一步的 ACD 风险蛋白进行交叉，筛出真正有可能处在因果链中段的中介蛋白。

## 第七步：中介分析，寻找连接 DJD 与 ACD 的关键蛋白

### 研究目的

这是全文机制部分的收束步骤，目的是回答：

- 哪些蛋白真正位于 DJD 和痴呆之间？
- 它们能够解释 DJD 增加痴呆风险的多大比例？

### 研究步骤

作者把三类蛋白筛选结果做交叉整合：

- 与 DJD 显著相关的蛋白
- 与 ACD 显著相关的蛋白
- 中介分析中达到统计学标准的候选蛋白

然后使用 mediation 分析估计每个蛋白对“DJD -> ACD”路径的中介比例，并用 1000 次 bootstrap 计算 95% 置信区间。

### 研究结果

最终识别出 18 个显著中介蛋白。最关键的几个如下：

- HAVCR1：中介比例 52.6%（95% CI: 39.1%-78.0%）
- GDF15：22.0%（95% CI: 14.5%-46.8%）
- COL6A3：14.3%（95% CI: 7.7%-28.4%）
- IGFBP4：11.6%
- CHGA：10.2%
- MMP12：10.1%
- LGALS4：9.5%
- KLK4：9.4%
- VSIG2：8.8%
- CHI3L1：8.5%
- ACE2：8.0%
- IL6：6.6%

作者进一步把这些蛋白按生物学功能归纳，大致可分成几类：

- 炎症与免疫相关：IL-6、MMP12、CHI3L1、HAVCR1
- 细胞黏附与信号调节相关：LGALS4、ADGRG1、CDCP1
- 生长因子与基质重塑相关：GDF15、IGFBP4、COL6A3
- 血管功能相关：ACE2

### 这一步说明了什么

这是全文最重要的机制结论。它说明：

- DJD 与痴呆之间并不是“直接跳跃式”关联
- 这条路径至少部分经过了系统性蛋白改变的中介
- 最核心的中介主题并不是经典 AD 的淀粉样蛋白通路，而是：
  - 全身炎症
  - 血管功能障碍
  - 细胞外基质与组织重塑

作者因此提出，DJD 可能通过慢性低度炎症、内皮功能损伤、血脑屏障破坏和血管相关脑损伤，推动痴呆尤其是 VaD 的发生。

## 4. 作者最终形成的研究闭环

把全文压缩成一条最清晰的研究主线，可以概括为：

1. 先在大样本前瞻性队列中发现：DJD 患者未来更容易发生痴呆，尤其是 VaD。
2. 再用多种敏感性分析、亚组分析和 PSM 证明：这一关联具有稳健性，不太像偶然结果。
3. 然后在蛋白组层面证明：DJD 会引起系统性循环蛋白改变。
4. 接着证明：其中不少蛋白本身也与未来 ACD 风险相关。
5. 最后通过中介分析说明：18 个关键蛋白构成了 DJD 影响痴呆风险的重要中间桥梁。

因此，这篇文章的完整研究框架可以概括为一句话：

> 作者先用大规模队列流行病学证实“DJD 与痴呆风险相关”，再用血浆蛋白组学解释“这种相关为何发生”，最终把机制聚焦到炎症、血管损伤和细胞外基质重塑三条主线。

## 5. 文章的核心结论

基于正文内容，这篇文章最终传达了 5 个最关键的信息：

1. DJD 是痴呆，尤其是血管性痴呆的重要风险标志之一。
2. 这种关联在调整人口学、生活方式和共病因素后仍然存在。
3. 结果经过敏感性分析和 PSM 复核后仍然稳健。
4. DJD 并非局部关节问题，而伴随明显的全身蛋白组改变。
5. HAVCR1、GDF15、COL6A3、IL6、ACE2 等蛋白可能是连接关节退变与认知衰退的关键中介。

## 6. 如果把这篇文章当作“研究设计模板”，它的方法学价值在哪里

这篇文章的方法学示范意义也很强，主要体现在：

- 先用大队列建立稳健关联
- 再用多重敏感性分析排查偏倚
- 再用 PSM 做类重复验证
- 最后把组学数据引入中介分析，完成“现象 -> 机制”的闭环

所以，它不是一篇只做“相关性报告”的文章，而是一篇试图把**流行病学发现**和**分子机制解释**串成一条完整证据链的研究。
