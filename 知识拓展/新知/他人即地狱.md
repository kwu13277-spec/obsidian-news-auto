# “他人即地狱”：萨特《禁闭》中的存在主义困境与辩证解读

内容类型：主题知识介绍

## 摘要

“他人即地狱”出自萨特剧本《禁闭》，揭示人际关系中因凝视、自欺和主体性争夺而陷入的精神封闭状态。本文说明其核心含义，分析对普通人的启示，并强调需要辩证看待他人与自我的关系。

## 关键词

他人即地狱；萨特；《禁闭》；存在主义；凝视；自欺；主体性冲突

### 1. 出处：萨特独幕剧《禁闭》

本节概要：明确“他人即地狱”的文本来源，说明剧情框架和该句在剧中的位置。

- 该句出自法国存在主义哲学家让‑保罗·萨特（Jean‑Paul Sartre）1944年发表的独幕剧《禁闭》（法文原名 *Huis clos*，常译为《密室》或《间隔》）。
- 剧中场景设定为一间没有窗户、没有镜子、永远亮着灯的第二帝国风格客厅，三个死者被先后带入房间：逃兵加尔散、女同性恋伊内丝、上流社会贵妇艾丝黛尔。
- 在剧情推进中，三人相互审视、相互依赖、相互折磨，无法逃离彼此的注视。剧本最后，加尔散在绝望中喊出：“他人即地狱！”（*L’enfer, c’est les autres.*）
- 该句并非泛指所有人际关系都必然痛苦，而是在具体情境下揭示一种极端的生存困境。《禁闭》的情节其实非常简单，甚至可以说“没有传统意义上的剧情”。因为整部剧的场景只有一个——那个没有窗户、没有镜子、永远亮着灯的第二帝国风格客厅，而三个角色从始至终都在这个封闭空间里互动。

以下是主要情节脉络：

### 1. 入场：三个死者先后被带入

- **加尔散**最先被带入房间。他是一个记者、文人，生前在战争中试图逃往墨西哥，被抓后枪决。他一直担心别人把自己视为懦夫。
- 接着**伊内丝**被带入。她是邮政职员，一个女同性恋者。生前她与表兄的妻子同居，导致表兄自杀，最后她在与情人一同开煤气自杀时身亡。她坦率、敏锐且极具攻击性。
- 最后是**艾丝黛尔**。她是一个年轻貌美的上流社会贵妇，生前为了掩盖私情溺死了自己的私生女，导致情人自杀，自己后来死于肺炎。她极度虚荣，完全依赖男性的注视来确认自身的存在。

### 2. 相互揭示：生前的罪过被逐一暴露

三人最初都试图维持体面的伪装，把自己打扮成无辜的受害者。他们一开始还默契地回避谈论各自来地狱的原因，装出文明和客气的样子。

但**伊内丝**是第一个打破虚伪平静的人。她拒绝假装友好，尖锐地迫使每个人坦白自己的罪行：

- 她看出**加尔散**并非为正义而死，而是因为想当逃兵。
- 她戳穿**艾丝黛尔**只是想用美貌引诱男人，而非真心忏悔。
- 她自己也坦承是“一盏让人燃烧的灯”，以折磨他人为乐。

在伊内丝的逼迫下，三人层层剥开彼此的伪装，生前的罪恶与自欺被完全暴露于他人眼中。

### 3. 三角折磨：不可能的欲望与永恒的囚禁

罪过暴露后，三人之间形成了一种**相互依赖又相互折磨的三角关系**：

- **加尔散**想要伊内丝的认同：他对艾丝黛尔的肉体毫无兴趣，他唯一渴望的是伊内丝（这个最冷酷的审视者）能告诉自己“你不是懦夫”。他的自由取决于一个不肯给他认可的他人。
- **艾丝黛尔**想要加尔散的爱：她对伊内丝充满憎恶，只想用自己的女性魅力占有加尔散。她不在乎加尔散是谁，只要他愿意扮演“注视她美貌的男人”就行。
- **伊内丝**想要艾丝黛尔：她对加尔散毫无欲望，甚至视他为自己与艾丝黛尔之间的障碍。她不断戳穿加尔散的自欺，同时向艾丝黛尔表露爱意，但艾丝黛尔厌恶她的同性倾向。

于是，每当两个人试图靠近，另一个人就会成为障碍。**伊内丝是加尔散的“地狱”**——她永远拒绝给他想要的认可，还用他生前懦弱行为的细节持续折磨他。**加尔散是艾丝黛尔的“地狱”**——她需要男性的爱来确认自己的魅力，但加尔散对此毫不配合。**艾丝黛尔是伊内丝的“地狱”**——她永远得不到艾丝黛尔的爱，却要眼睁睁看着艾丝黛尔不断投向加尔散。

### 4. 房门开了，却无人离开

在剧情高潮时，房门突然打开了。按照常理，这三人本可以走出房间，逃离彼此。

但没有人走。

- **加尔散**不敢走：他需要留下来，直到说服伊内丝他不是一个懦夫。如果没有伊内丝的认可，走出这扇门，他仍然背负着“懦夫”的标签。
- **艾丝黛尔**不敢走：没有加尔散的爱，去哪里都没有意义。
- **伊内丝**不敢走：她离不开艾丝黛尔，哪怕被厌恶也要留在她身边。

他们明白了地狱的真相：那扇门通向任何地方，“任何地方”都是“别的地方”。但地狱不是物理空间的囚禁，而是彼此注视的永不可逃。

### 5. 结局：笑声中的永恒

最终，三人都意识到他们的处境是永恒的——没有人会入睡，没有人会闭眼，没有人会放弃审视他人。他们将在永远亮着的灯光下，永远保持清醒，永远相互折磨。

加尔散在绝望中大笑，说出那句著名的台词：

> “那么，让我们继续吧。”  
> （长久的寂静之后）  
> “他人即地狱！”

全剧就在这绝望的笑声与对永恒的接受中结束。没有幕布落下，没有死亡可以期待，只有三个意识在相互地狱中无休无止地共存下去。

### 2. 核心含义：人与人之间的三重困境

本节概要：解释“他人即地狱”所揭示的三种存在主义困境——凝视的对象化、自欺的依赖性和主体性的争夺。

- **凝视与对象化**：他人的注视会将“我”凝视为一个客体、一个标签化对象。我不再是拥有无限可能的主体，而被迫以他人眼中的固定形象存在。比如被贴上“懦夫”“荡妇”“虚伪者”等标签后，仿佛自己的全部就被这些定义所穷尽，失去了重新定义自我的自由。
- **自欺与他人的共谋**：人往往通过他人的判断来逃避自身的自由与责任，这种“自欺”使得他人的评价变成一种牢笼。在剧中，加尔散反复乞求伊内丝告诉自己“你不是懦夫”，这种对他者认可的病态依赖，恰恰使他人成为自己精神痛苦的原因。
- **主体性的零和争夺**：三个角色都在争夺对当下情境的叙事主导权，每个人都要成为关系中的主体，拒绝成为他人的客体。由于欲望和需求彼此冲突，且无法达成真正的共同自由，这种争夺使他们永远相互折磨。

<a id="section-3"></a>

### 3. 对普通人的启示

本节概要：从日常人际关系角度，说明该理念可以带来的自我审视和价值。

- **警惕被他人评价绑架**：过度在意他人眼光，会让人转而按照他人的期待塑造自己，压抑真实的意愿和潜能。认识到他人的注视可能带来束缚，有助于保持自我决定的能力。
- **审视自己对他人的依赖**：当一个人过度寻求他人的认可、同情或评价时，实际上是在放弃自己的自由，把定义自己的权力交给了他人。这种依赖会制造精神上的“地狱感”。
- **人际关系中的权力与尊重**：日常交往中，无视对方的主体性、强行定义对方、把对方工具化，都会造成类似“地狱”的隐性暴力。启示人们尊重他人的自由，而非试图成为他人生活的绝对评判者。

<a id="section-4"></a>

### 4. 辩证看待：他人并非绝对地狱

本节概要：指出不能将该哲学命题泛化为对一切人际关系的否定，而应看到他人的建构性作用。

- **萨特本人的澄清**：萨特在公开访谈中多次强调，这句话并不是指人与人之间的关系总是糟糕透顶，而是说当人际关系被扭曲、被操纵、被客体化时，他人才会变成地狱。正常健康的交往并不落入这一困境。
- **他人是自我认知的必要参照**：没有他人的存在，人甚至无法获得完整的自我意识。如同剧中缺少镜子，人需要通过他人的反应来认识自己。他人的反馈在多数情况下帮助个体校准自我认知，而非仅带来痛苦。
- **合作与共同自由的积极可能**：萨特在后期哲学中强调群体实践中“共同自由”的可能。当个体不以占有和控制为目的，而是共同指向某个目标时，他人可以从地狱变为共同行动者，构建相互成就的关系。
- **实用区分**：对普通人而言，关键在于区分“健康的他人反馈”与“病态的他者依赖”。前者是平等主体的交流，后者则是放弃自身主体性的自我异化。把他人简单地一概视作地狱，同样是放弃批判思考的表现。
