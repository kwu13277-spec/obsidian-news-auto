## 正文

### 1. 从单中介到高通量筛查：问题的提出

本节概要：本节说明高通量蛋白中介筛查试图解决什么问题，以及它和传统中介分析的根本区别。

- **传统中介分析**：研究者有**一个**候选中介变量（如某个特定蛋白），检验它是否介导了暴露对结局的影响。
  - 问题明确、假设驱动、检验数量单一。
  - 例："CRP 是否介导了 BMI 对心血管事件的影响？"

- **高通量蛋白中介筛查**的研究场景：研究者有**数千个**蛋白的测量数据，没有一个先验假设，需要在全蛋白组范围内系统性地**搜索哪些蛋白可能是中介**。
  - 例："在 3000 种血浆蛋白中，哪些介导了 BMI 对冠心病风险的影响？"
  - 特点：数据驱动、规模大、没有单一先验假设。

- **为什么需要这种方法**：
  - 蛋白组学技术（SomaScan、Olink、MS 等）已成熟，一次可测量数千种蛋白。
  - 生物机制往往是多条通路共同作用，单变量中介分析视野太窄。
  - 筛查结果可以产生新的机制假说，为后续实验提供靶点。

- **高通量蛋白中介筛查的核心问题**：
  - 如何在数千个候选蛋白中**同时**做中介检验？
  - 如何控制多重比较带来的假阳性？
  - 蛋白之间的相关性如何处理？
  - 可重复性如何评估？

### 2. 蛋白组学数据的特点与挑战

本节概要：本节介绍高通量蛋白数据的关键特征，这些特征直接决定了中介分析方法的选择。

#### 2.1 数据维度

- **高维度**：蛋白数量（p）数百到数千，甚至上万。
- **样本量**（n）通常几十到几千，存在"p 接近 n"甚至"p > n"的情况。

#### 2.2 数据特征

| 特征 | 具体表现 | 对中介分析的影响 |
|------|---------|----------------|
| **高相关性** | 同一通路的蛋白高度共表达 | 违反传统中介分析的中介变量独立性假设 |
| **测量误差** | 蛋白定量存在技术噪声（批次效应、平台差异） | 导致效应估计向零衰减（attenuation bias） |
| **缺失数据** | 部分蛋白低丰度无法检出 | 处理不当引入偏倚 |
| **偏态分布** | 蛋白浓度通常呈右偏 | 需要对数转换或非参数方法 |
| **尺度差异** | 不同蛋白的浓度范围相差数个数量级 | 需要标准化 |

#### 2.3 样本约束

- 多数队列研究的蛋白组数据来自基线单一时间点的测量，难以建立时间先后顺序。
- 结局变量和蛋白同时测量时，因果方向无法确定（反向因果问题）。

### 3. 经典中介分析回顾

本节概要：快速回顾 Baron & Kenny 方法和因果逐步检验框架，为理解高通量扩展奠定基础。

#### 3.1 Baron & Kenny 四步法

对于暴露 $X$、中介 $M$、结局 $Y$：

1. $Y = cX + \epsilon_1$ → $c$（总效应，total effect）必须显著
2. $M = aX + \epsilon_2$ → $a$（暴露→中介路径）必须显著
3. $Y = c'X + bM + \epsilon_3$ → $b$（中介→结局路径）必须显著
4. 如果 $c'$ 不显著 → 完全中介；$c'$ 减小但显著 → 部分中介

#### 3.2 间接效应与 Sobel 检验

- **间接效应**（indirect effect）= $a \times b$
- **总效应** = 直接效应（$c'$）+ 间接效应（$a \times b$）
- **Sobel 检验**：检验 $H_0: a \times b = 0$，利用 delta 方法计算标准误

#### 3.3 Bootstrap 法

- Sobel 检验假定 $a \times b$ 的抽样分布为正态，在中小样本中常不成立。
- Bootstrap 法（非参数重抽样）直接估计 $a \times b$ 的置信区间，不依赖正态假设。
- **当前推荐**：bootstrap 置信区间法，优于 Sobel 检验。

#### 3.4 经典方法在高通量场景下的局限

- 每个候选蛋白都要运行三步回归 → 计算量线性增长（尚可接受）。
- 但每条路径的多重检验问题被放大 3 倍（$a$、$b$、$a \times b$ 各自一次）。
- 无法有效处理蛋白之间的相关性。
- Bootstrap 在数千个蛋白上逐一轮回计算量巨大。

### 4. 高通量中介筛查的核心策略

本节概要：本节说明高通量中介筛查的总体思路和关键选择节点。

#### 4.1 筛查 vs 验证的区分

- **第一阶段：筛查（screening）**
  - 对所有候选蛋白逐一做中介检验，得到每个蛋白的间接效应 p 值。
  - 输出：一个按显著性排序的候选蛋白列表。
  - 目的：不遗漏，优先保证敏感性。

- **第二阶段：验证（validation）**
  - 在独立队列中对筛查显著的蛋白进行定向验证。
  - 使用更保守的方法（如 bootstrap CI）。
  - 目的：控制假阳性，确保可重复性。

#### 4.2 两种主流策略

| 策略 | 核心思想 | 优点 | 缺点 |
|------|---------|------|------|
| **逐一遍历 + 多重校正** | 对每个蛋白独立做中介检验，然后用 FDR/Bonferroni 校正 | 简单直接，方法成熟 | 忽略蛋白间相关性和竞争中介问题 |
| **高维联合方法** | 一次建模所有中介变量（正则化回归或贝叶斯方法） | 能同时处理多个中介和相关性 | 计算复杂，解释性较差 |

#### 4.3 关键决策点

- **是否需要调整协变量**？年龄、性别、批次等需要在所有模型中校正。
- **间接效应是否显著**？使用 Sobel、bootstrap 还是乘积分布法？
- **多重校正的阈值**？FDR < 0.05 还是更严格的标准？
- **是否做 Bonferroni 进一步收紧**？取决于筛查的目的是发现还是验证。

### 5. 高维中介分析的统计方法分类

本节概要：本节系统梳理高通量中介分析的主要统计方法及其适用条件。

#### 5.1 方法一：逐一遍历 + Sobel/bootstrap 检验（Huang et al. 2018）

- **步骤**：
  1. 对每个蛋白 $k$，拟合三个回归：
     $M_k = a_k X + \epsilon$
     $Y = c'X + b_k M_k + \epsilon$
  2. 计算间接效应 $\hat{a}_k \hat{b}_k$ 及其标准误（delta method 或 bootstrap）。
  3. 计算 $p$ 值或置信区间。
  4. 对 $p$ 值做 FDR 校正（Benjamini-Hochberg）。

- **适用场景**：蛋白数量可控（<2000）、样本量 > 200、需要快速筛查。
- **R 包**：`mediation`、`mma`

#### 5.2 方法二：高维中介分析（HIMA, Zhang et al. 2016）

- **核心思想**：先用 Penalized regression（LASSO）筛选出可能的中介变量，再对筛选后的子集做传统中介检验。

- **步骤**：
  1. 联合模型：$Y = X\beta + M\gamma + \epsilon$，用 LASSO 对 $\gamma$ 做变量选择（压缩大部分 $\gamma_k$ 为 0）。
  2. 通过交叉验证选择惩罚参数 $\lambda$，保留非零系数的蛋白。
  3. 对保留下来的蛋白逐一做经典中介分析（检验 $a_k b_k = 0$）。
  4. 多重校正（FDR）。

- **优点**：
  - 能处理 $p > n$ 的高维数据。
  - 变量选择步骤过滤掉大量无效蛋白，减轻多重检验负担。
  - 对相关蛋白有一定容受性（LASSO 倾向于在相关变量中随机选择一个）。

- **缺点**：
  - LASSO 的变量选择稳定性不足——对数据扰动敏感。
  - 高度相关的蛋白可能只有一个被选出，漏掉真正的共中介。
  - 两步法导致最终推断的方差被低估（需要特殊的标准误估计）。

- **R 包**：`HIMA`

#### 5.3 方法三：联合显著性检验（Joint Significance Test, JST）

- 不计算 $a \times b$ 的标准误，而是直接检验两个条件：
  1. $a_k$ 的 p 值 < $\alpha_1$
  2. $b_k$ 的 p 值 < $\alpha_2$

- 如果两个条件同时满足 → 判定为显著中介。

- **优点**：不需要计算复杂的 $a \times b$ 标准误，计算简单。
- **缺点**：无法量化间接效应大小，没有置信区间。

#### 5.4 方法四：Mediation Test for High-dimensional Omics（MTO, Zhang 2022）

- 对条件方法进一步改进：使用 $p_a$ 和 $p_b$ 的联合分布作为检验统计量。
- 比 JST 有更好的控制错误率表现。

#### 5.5 方法五：贝叶斯高维中介分析

- 对 $a$ 和 $b$ 系数施加 spike-and-slab 先验，后验概率 > 阈值即判为中介。
- **优点**：自然处理不确定性、提供后验包含概率（容易解释）。
- **缺点**：计算量大、MCMC 收敛诊断在高维下困难。
- **工具**：`BMMA`（Bayesian Mediation Mixture Analysis）

#### 5.6 方法对比总表

| 方法 | p > n 处理能力 | 处理蛋白相关 | 计算速度 | 输出 | 推荐样本量 |
|------|--------------|------------|---------|------|-----------|
| 遍历+Sobel | ❌ 不可 | ❌ 差 | 快 | p 值、效应量 | > 500 |
| 遍历+bootstrap | ❌ 不可 | ❌ 差 | 中 | CI | > 200 |
| HIMA（LASSO筛选） | ✅ 可 | 中 | 中 | p 值 | > 100 |
| Joint Significance | ✅ 可 | ❌ 差 | 极快 | 二元判断 | > 50 |
| MTO | ✅ 可 | 中 | 快 | p 值 | > 100 |
| 贝叶斯方法 | ✅ 可 | ✅ 好 | 慢 | 后验概率 | > 100 |

### 6. 多重假设校正：在数千个检验中控制错误

本节概要：本节说明为什么高通量中介分析比普通差异蛋白分析面临更复杂的多重检验问题。

#### 6.1 问题的规模

- 如果有 3000 个蛋白，每个蛋白做一次间接效应检验 → 3000 个零假设。
- 如果每个检验使用 $\alpha = 0.05$，即使**没有任何真正的中介**，也会有约 $3000 \times 0.05 = 150$ 个"显著"结果。

#### 6.2 常用校正方法

| 方法 | 控制指标 | 严格程度 | 高通量中介适用性 |
|------|---------|---------|----------------|
| Bonferroni | 家庭错误率（FWER） | 极严格 | 用于验证阶段，或蛋白数较少（<50） |
| Benjamini-Hochberg（BH） | 错误发现率（FDR） | 中等 | **推荐用于筛查阶段** |
| q-value（Storey） | 阳性发现错误率（pFDR） | 中等偏宽松 | 适合蛋白数极大（>5000） |
| BY（Benjamini-Yekutieli） | 任意依赖下的 FDR | 严格 | 当已知蛋白高度相关时用 |

#### 6.3 高通量中介特有的多重检验问题

- **双重检验问题**：如果对 $a$ 路径和 $b$ 路径分别做多重校正，会过度严格。推荐只对**间接效应**（$a \times b$）的 p 值做一次多重校正。

- **候选筛选后的再检验问题**：如果在 LASSO 筛选后对保留下来的蛋白做经典中介检验，这些检验本身不需要额外的多重校正——因为筛选步已经压缩了检验数量。但这是有争议的（筛选步引入了 selection bias）。

- **建议流程**：
  - 筛查阶段：计算所有蛋白间接效应 p 值 → BH-FDR 校正 → FDR < 0.2 或 0.1（用较宽松阈值）
  - 验证阶段：在独立数据中，只对 FDR 显著的蛋白做 bootstrap 检验 → $\alpha = 0.05$

### 7. 分析流程总览

本节概要：本节给出一个从数据准备到结果报告的可操作分析流程。

#### 7.1 流程图

```
┌────────────────────────────────────────┐
│ 1. 数据准备                            │
│    - 暴露变量 X（如 BMI、基因风险评分）     │
│    - 结局变量 Y（如疾病状态、生存时间）     │
│    - 蛋白矩阵 M（~3000 蛋白）              │
│    - 协变量 C（年龄、性别、批次等）         │
└─────────────────┬──────────────────────┘
                  ▼
┌────────────────────────────────────────┐
│ 2. 数据预处理                          │
│    - 蛋白表达标准化（log2 转换 + Z-scaling)│
│    - 缺失值过滤（剔除缺失率 > 20% 的蛋白）│
│    - 残差化处理（去除协变量影响，可选）    │
└─────────────────┬──────────────────────┘
                  ▼
┌────────────────────────────────────────┐
│ 3. 高通量中介筛查                       │
│    - 选择方法（遍历/HIMA/JST/MTO）      │
│    - 逐蛋白拟合 a 和 b 路径             │
│    - 计算间接效应及其 p 值               │
└─────────────────┬──────────────────────┘
                  ▼
┌────────────────────────────────────────┐
│ 4. 多重假设校正                        │
│    - 对间接效应 p 值做 BH-FDR 校正      │
│    - 设定 FDR 阈值（0.05-0.2）          │
│    - 列出显著候选蛋白                    │
└─────────────────┬──────────────────────┘
                  ▼
┌────────────────────────────────────────┐
│ 5. 验证与敏感性分析                     │
│    - 在独立队列中验证                    │
│    - Bootstrap 置信区间                 │
│    - 置换检验（permutation test）      │
│    - 反向因果检验（暴露 ↔ 中介的排序）    │
└─────────────────┬──────────────────────┘
                  ▼
┌────────────────────────────────────────┐
│ 6. 结果解释与报告                       │
│    - 提供效应量（间接效应、中介比例）      │
│    - 生物学注释（通路富集、PPI）         │
│    - 敏感性分析结果                      │
│    - 可重复性评估                        │
└────────────────────────────────────────┘
```

#### 7.2 各步骤要点

- **步骤 2 预处理**：标准化非常关键——不同蛋白的尺度差异会导致间接效应不可比。
- **步骤 3 方法选择**：如果样本量 > 300 且蛋白数 < 2000，遍历 + bootstrap 简单可靠。如果 p 接近 n 或 p > n，优先考虑 HIMA。
- **步骤 5 验证**：始终在独立样本中验证。如果只有一个数据集，可使用交叉验证或样本分割。

#### 7.3 中介比例计算

对于显著的蛋白中介：
$$
\text{Mediation proportion} = \frac{a \times b}{c} \times 100\%
$$

其中 $c$ 是暴露对结局的总效应（仅含 $X$ 的模型）。

注意：当存在多个中介时，各中介比例之和可能超过 100%（因为中介路径存在重叠），这是正常现象。

### 8. 常用软件与工具

本节概要：本节列出进行高通量蛋白中介筛查可用的主流 R 包和软件。

| 工具 | 方法 | 特色 | 适用 |
|------|------|------|------|
| `mediation` | 遍历 + bootstrap | 最经典，支持多种效应尺度 | 单中介或少量中介验证 |
| `mma` | 遍历 + 多种检验 | 支持多个中介同时建模 | 中等规模中介筛选 |
| `HIMA` | LASSO + 中介检验 | 专为高维中介设计 | p > n 场景 |
| `medflex` | 基于自然效应模型 | 可以处理暴露-中介交互 | 有交互项的场景 |
| `CMAverse` | 统合多种中介方法 | 可视化好、报告完整 | 整体分析流程 |
| `mediation.test` | MTO 方法 | 快速、适合大规模筛查 | 蛋白数 > 5000 |
| `BMMA` (Bayesian) | 贝叶斯中介混合模型 | 提供后验概率 | 小样本、需要概率推断 |

**推荐组合**：

- 筛查阶段：`HIMA` 或 `mediation.test`
- 验证阶段：`mediation`（bootstrap 选项）
- 可视化：`CMAverse`（结果展示）

### 9. 应用实例与典型论文

本节概要：本节通过一个典型分析场景说明高通量蛋白中介筛查的实际应用。

#### 9.1 典型研究场景

**研究问题**：BMI 的哪些蛋白中介介导了它对冠心病（CHD）风险的影响？

**数据**：
- 暴露（X）：BMI
- 中介（M）：约 3000 种血浆蛋白（SomaScan 平台）
- 结局（Y）：冠心病发生（二元：是/否）
- 样本量：训练集 1000 人，验证集 800 人
- 协变量：年龄、性别、吸烟、饮酒

**分析步骤**：

1. **标准化**：每种蛋白做 log2 转换 + Z-score 标准化
2. **缺失过滤**：去除缺失率 > 20% 的蛋白（剩余约 2800 种）
3. **高通量筛查**：
   ```r
   library(HIMA)
   hima_result <- hima(
     X = dat$bmi,          # 暴露
     Y = dat$chd,          # 结局（二元）
     M = protein_matrix,   # 蛋白矩阵（2800列）
     covar = covar_matrix, # 协变量
     penalty = "MCP",      # MCP惩罚
     topN = NULL,          # 不预设候选数量
     FDR = 0.1
   )
   ```
4. **结果**：
   - HIMA 筛选出 15 个候选蛋白
   - 其中 8 个 FDR < 0.1
   - 间接效应最大的是 IL-6（$a = 0.32, b = 0.18$, 间接效应 = 0.058）
   - 中介比例：IL-6 介导了 BMI→CHD 总效应的 12.4%

5. **验证**：
   - 在验证队列中对 8 个蛋白用 `mediation` 包做 bootstrap 检验
   - 5 个蛋白的中介效应 bootstrap CI 不包含 0 → 验证成功

6. **生物学解释**：
   - IL-6、CRP、TNFRSF1A 等属于**炎症通路**
   - ADIPOQ 属于**脂肪因子信号**
   - 这些结果提示 BMI 通过炎症和脂肪因子通路影响冠心病风险

### 10. 局限性与注意事项

本节概要：本节说明高通量蛋白中介筛查的主要局限，以及如何避免常见陷阱。

#### 10.1 方法与假设限制

| 局限 | 说明 | 缓解策略 |
|------|------|---------|
| **横断面中介** | 蛋白和结局在同一时间点测量，因果方向不确定 | 使用时间滞后数据、孟德尔随机化作为工具变量 |
| **未测量的混杂** | 暴露-中介、中介-结局之间存在未测量混杂 | 敏感性分析（如 E-value）、负对照分析 |
| **中介-结局混杂受暴露影响** | 在传统框架下几乎不可直接解决 | 使用逆概率加权或边缘结构模型 |
| **测量误差** | 蛋白定量误差使间接效应趋向于零 | 多重 imputation、误差校正模型 |
| **选择偏倚** | 删失或选择入组与暴露或中介相关 | 加权方法、敏感性分析 |

#### 10.2 多个相关的蛋白中介

- 当一个通路中多个蛋白高度相关时：
  - 逐一遍历法会相互竞争，相关性可以掩盖真实的中介信号。
  - HIMA 的 LASSO 步可能随机选择其中一个。
  - **建议**：对显著中介蛋白做通路富集，而不是过度解释单个蛋白。

- **后续分析**：对筛选出的显著蛋白做网络分析（PPI、KEGG 通路），识别是否有共同的生物学主题。

#### 10.3 统计与可重复性

- **FDR 阈值的选择**本质上是主观的。FDR < 0.2 在筛查阶段可接受，但验证阶段应使用 $\alpha = 0.05$。
- **过拟合风险**：两步法（先筛选、后检验）容易过拟合训练数据，验证集始终必不可少。
- **可重复性评估**：
  - 在多个独立队列中重复。
  - 使用相同的蛋白平台（不同平台的蛋白定量可能系统差差异大）。

#### 10.4 给初学者的建议

- **先跑一遍完整的单一中介分析**：选一个你最感兴趣的蛋白，走通所有步骤，确认理解每个输出的含义。
- **从简单的遍历 + BH 校正开始**：刚入门的分析不需要复杂的 HIMA 或贝叶斯方法，遍历方法更容易理解和调试。
- **别忽略预处理**：标准化和缺失值处理对结果的影响可能比分析方法本身更大。
- **验证比发现更重要**：高通量筛查的目的不是"报告 P < 0.05 的蛋白"，而是找到可以被验证的真正中介。
- **结果解释保持克制**：高通量中介分析提示的是**统计中介**（statistical mediation），不是**因果中介**（causal mediation）。
