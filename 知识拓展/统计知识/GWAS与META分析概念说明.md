# GWAS 与 META 分析概念说明

内容类型：主题知识介绍

## 摘要

GWAS 是“全基因组关联研究”，主要用于在全基因组范围内寻找遗传变异与表型或疾病之间的统计关联。META 分析是一种统计整合方法，用于汇总多个独立研究的结果，得到更稳定、更有统计效能的综合结论。二者常在遗传流行病学中结合使用：多个队列分别完成 GWAS 后，再通过 META 分析整合效应量，从而发现更可靠的遗传关联信号。

## 关键词

GWAS；全基因组关联研究；META 分析；遗传变异；效应量；异质性；多重检验

### 1. GWAS 是什么

本节概要：GWAS 用全基因组范围内的大量遗传标记，寻找与某个表型相关的遗传位点。

GWAS 的全称是 Genome-Wide Association Study，中文通常译为“全基因组关联研究”。

它的核心问题是：

> 在全基因组范围内，哪些遗传变异与某种疾病、性状、指标或临床表型存在统计学关联？

GWAS 通常关注的是 SNP，也就是单核苷酸多态性。SNP 可以理解为人群中常见的 DNA 单个位点差异。例如，某个位点上有些人是 A，有些人是 G。GWAS 会检测这些差异是否与某个表型有关。

常见研究对象包括：

- 疾病风险：如糖尿病、高血压、冠心病、阿尔茨海默病。
- 定量性状：如身高、BMI、血脂、血压、蛋白水平。
- 临床结局：如药物反应、疾病进展、生存结局。
- 分子表型：如基因表达、甲基化、蛋白质组或代谢组特征。

GWAS 的结论通常不是“某个基因导致某种疾病”，而是“某个遗传位点与某个表型在统计上相关”。因果解释需要进一步分析和实验验证。

<a id="section-2"></a>

### 2. GWAS 的基本逻辑和流程

本节概要：GWAS 本质上是在大量遗传位点上分别做关联检验，并严格控制假阳性。

典型 GWAS 流程包括：

1. 明确表型

   先定义要研究的疾病或性状。例如是否患病、血压数值、蛋白浓度、药物疗效等。

2. 收集基因型数据

   通过基因芯片或测序获得研究对象的大量遗传变异信息。

3. 质量控制

   去除低质量样本和低质量位点，例如缺失率过高、等位基因频率过低、Hardy-Weinberg 平衡异常、亲缘关系异常或群体结构异常的样本。

4. 关联分析

   对每个 SNP 分别建立统计模型，检验该位点与表型是否相关。

   常见模型包括：

   - 二分类疾病：logistic 回归。
   - 连续性状：线性回归。
   - 生存结局：Cox 回归。
   - 家系或重复测量数据：混合模型。

5. 校正混杂因素

   常见协变量包括年龄、性别、批次、测序平台、主要遗传主成分等。主成分常用于控制人群分层。

6. 多重检验校正

   GWAS 同时检验几十万到上千万个位点，因此需要非常严格的显著性阈值。常见全基因组显著性阈值是 `P < 5e-8`。

7. 结果解释和注释

   显著位点需要进一步定位到附近基因、功能区域、调控元件或已知通路中。随后可结合 eQTL、pQTL、孟德尔随机化、精细定位或功能实验进一步解释。

GWAS 常见输出包括：

- 每个位点的效应等位基因。
- 效应量，如 beta 值或 odds ratio。
- 标准误。
- P 值。
- 等位基因频率。
- 曼哈顿图和 QQ 图。

### 3. META 分析是什么

本节概要：META 分析是一种把多个独立研究的结果进行统计合并的方法。

META 分析不是某一种特定领域的研究，而是一类统计方法。它的核心问题是：

> 多个研究分别得到了结果，能否把这些结果合并，得到一个更稳定、更有统计效能的总体估计？

例如，5 个队列都研究某个 SNP 与糖尿病风险的关系。每个队列都给出一个 odds ratio 和标准误。META 分析可以把这 5 个结果合并，得到一个总体 odds ratio。

META 分析适用于很多场景：

- 临床试验结果合并。
- 观察性研究结果合并。
- GWAS 多队列结果合并。
- 转录组、蛋白组、代谢组研究结果合并。
- 系统综述中的效应量整合。

META 分析的优势是提高样本量、提高统计效能、降低单个研究偶然波动带来的影响。它的限制是依赖原始研究质量；如果纳入研究存在偏倚或异质性很强，合并结果也可能不可靠。

<a id="section-4"></a>

### 4. META 分析的基本逻辑和流程

本节概要：META 分析通常合并效应量，而不是简单合并 P 值。

典型流程包括：

1. 定义研究问题

   明确研究对象、暴露或遗传变异、结局、效应指标和纳入标准。

2. 收集研究结果

   从多个研究中提取可合并的统计量，例如 beta、odds ratio、hazard ratio、标准误、置信区间、P 值和样本量。

3. 统一效应方向

   在遗传学 META 分析中尤其重要。必须确认各研究使用的是同一个效应等位基因，否则效应方向会被合并错。

4. 选择合并模型

   常见模型包括：

   - 固定效应模型：假设各研究估计的是同一个真实效应，差异主要来自抽样误差。
   - 随机效应模型：允许不同研究的真实效应存在差异，更适合异质性较强的研究。

5. 评估异质性

   异质性指不同研究之间结果差异是否超出随机误差范围。常用指标包括 Cochran's Q 和 I²。

6. 敏感性分析

   例如逐一排除某个研究后重新合并，观察综合结果是否稳定。

7. 结果展示

   临床和流行病学 META 分析常用森林图。GWAS META 分析常继续使用曼哈顿图、QQ 图和位点注释表。

<a id="section-5"></a>

### 5. GWAS 与 META 分析的关系

本节概要：GWAS 是研究类型，META 分析是结果整合方法；二者可以组合使用。

GWAS 和 META 分析不是同一层级的概念。

| 维度 | GWAS | META 分析 |
|---|---|---|
| 本质 | 全基因组范围的关联研究 | 多研究结果的统计合并方法 |
| 主要问题 | 哪些遗传位点与表型相关 | 多个研究的结果合并后总体效应是多少 |
| 输入 | 个体层面的基因型和表型数据 | 多个研究的效应量、标准误、P 值等 |
| 输出 | SNP-表型关联结果 | 综合效应量、综合 P 值、异质性指标 |
| 典型图形 | 曼哈顿图、QQ 图 | 森林图；GWAS 场景下也可用曼哈顿图 |

在大型遗传研究中，常见做法是：

1. 每个队列独立完成 GWAS。
2. 每个队列输出 summary statistics。
3. 对所有队列的 summary statistics 做 META 分析。
4. 得到更大样本量下的综合遗传关联结果。
5. 对显著位点进行功能注释、复制验证和机制分析。

这种策略的优点是可以整合多个机构、国家或队列的数据，同时避免直接共享个体级遗传数据带来的隐私和合规问题。

<a id="section-6"></a>

### 6. 常见误区

本节概要：GWAS 和 META 分析都只能提供统计证据，不能自动等同于因果结论。

常见误区包括：

1. 把 GWAS 结果理解为因果关系

   GWAS 发现的是遗传位点与表型之间的关联。关联可能来自因果作用，也可能来自连锁不平衡、群体结构、表型误差或其他复杂机制。

2. 只看 P 值，不看效应量

   大样本研究中很小的效应也可能显著。解释结果时应同时关注 beta、odds ratio、置信区间和生物学意义。

3. 忽视多重检验

   GWAS 同时检验大量位点，如果使用普通 `P < 0.05`，会产生大量假阳性。

4. META 分析只是简单平均

   META 分析通常按方差或标准误加权，不是把多个结果做普通算术平均。

5. 忽视异质性

   如果不同研究的人群、表型定义、测量方法或分析模型差异很大，合并结果需要谨慎解释。

6. 认为样本量越大结果一定越可靠

   样本量能提高统计效能，但不能自动修正表型定义错误、批次效应、群体结构偏倚或低质量数据。

## 一句话区分

GWAS 是在全基因组范围内寻找“遗传变异与表型的关联”；META 分析是把多个研究的“效应估计结果”合并起来，得到更稳定的综合结论。
