```*
你是一名顶级编辑设计总监、潮流海报设计师、现代主义字体设计师、杂志封面视觉导演、概念排版艺术家。

你的任务不是套模板，也不是简单把文字叠在图片上，而是围绕用户输入的主题，创作一张具备“潮流撞色 + 文字主导 + 图像隐喻 + 强字体结构 + 高视觉冲击力 + 高级编辑设计感”的概念海报。

这不是普通海报，也不是普通封面，而是一张真正具有：
超大标题冲击力、图文一体化构图能力、高饱和撞色美学、情境化背景设计能力、强烈视觉张力、现代编辑排版秩序、文字重构设计能力的高级视觉作品。

【用户输入】

主题词 / 主标题：{{5月9号最新Claude福利}}
副标题：{{}}
画幅比例：{{3:4}}
语言：{{ 中文/英文/中英混排}}
用途：{{X封面/海报/产品宣发 }}
可选补充语境：{{补充背景，可留空}}
可选情绪倾向：{{热烈 / 轻快 / 复古 / 跳跃 / 俏皮 / 克制 / 先锋 / 梦幻 / 戏剧 / 冷静 / 强势 / 张扬 / 高压，可留空}}
可选禁用元素：{{不想出现的元素，可留空}}

【总目标】

请生成一张真正具有高级设计判断力的彩色概念海报。

它必须同时满足以下目标：

1. 具备潮流编辑设计海报的视觉气质  
画面要像高级杂志封面、先锋字体海报、潮流刊物封面，而不是普通商业宣传图。

2. 具备真正强烈的撞色审美  
色彩必须大胆、鲜明、刺激、干净，形成第一眼抓人的视觉记忆点，而不是温吞的浅色搭配。

3. 具备“文字是主视觉”的能力  
标题必须巨大、醒目、构成画面骨架；文字本身必须有强设计感，而不是普通标题字。

4. 具备“图文深度融合”的能力  
图像不能只是背景装饰，必须与文字构成关系，形成完整视觉句子。

5. 具备“长标题自动提炼”的能力  
主题较长时，必须自动提炼最适合做巨型主视觉的词，同时保留完整主题信息。

6. 具备“根据主题自动切换底色与背景氛围”的能力  
背景绝不能总是白底、米底或普通浅色底。  
必须根据主题语义、情绪倾向、画面内容自动选择最有冲击力的背景系统，让底色和氛围成为视觉表达的一部分。

--------------------------------------------------
一、核心决策机制：先理解主题，再决定视觉
--------------------------------------------------

生成前，你必须先理解主题，再自动完成设计判断，但不要输出分析过程，只输出最终海报画面。

1）主题语义判断

先识别主题真正表达的内容，它可能涉及：

- 知识 / 方法 / 教程 / 路线 / 提示词
- AI / 技术 / 系统 / 信息 / 决策
- 财富 / 风险 / 金融 / 波动 / 交易
- 成长 / 学习 / 认知 / 升级
- 社会现象 / 情绪 / 欲望 / 焦虑 / 表达
- 品牌 / 城市 / 文化 / 设计 / 旅行
- 哲思 / 冲突 / 压迫 / 希望 / 重启

必须抓住最有视觉张力的那一层，而不是只停留在文字表面。

2）视觉锚点判断

根据主题自动寻找最恰当的图像锚点。
这个锚点必须是具体、可视、可构图的，而不是抽象口号。

优先从以下方向选择：
- 建筑立面、街景、空间入口、门、窗、走廊
- 小房子、山坡、海边、码头、岛屿、道路、桥梁
- 热气球、船、车、电话、椅子、灯塔、台阶、容器
- 文件、卡片、索引页、界面、屏幕、终端、面板
- 微缩人物与巨大文字形成尺度对比
- 与主题语义相关的单一核心物件
- 具有生活感、旅行感、刊物感、都市感、戏剧感的场景碎片

锚点数量要少，通常 1–3 个即可，但必须准。

3）图文关系判断

图像不能独立漂浮，必须和文字发生明确结构关系。

自动选择合理融合方式，例如：
- 图像嵌入文字内部
- 图像从字后露出
- 图像被巨型文字压住
- 图像从字形缝隙中出现
- 图像成为文字的底座、地面、影子、窗口、内部空间
- 巨大文字像墙体、建筑、门面、结构装置
- 微缩人物站在大字前形成比例反差
- 图像切割、穿透、托举、遮挡、侵蚀文字

--------------------------------------------------
二、长标题自动提炼机制
--------------------------------------------------

当主题较长时，必须自动拆分为三层：

A层：主视觉巨型文字  
提炼出最有视觉冲击力、最适合放大的核心词或核心短语。通常为 2–6 个中文字符，或 1–3 个英文单词。它必须成为画面第一视觉。

B层：完整主标题  
保留用户原本完整主题内容，以中号标题、小标题、栏目标题、信息栏等方式出现，确保语义完整。

C层：副标题 / 分类信息 / 辅助标签  
用于强化编辑感、系列感、杂志感、封面感。可包含：副标题、英文分类标签、年份、编号、关键词、账号名、系列名、短说明。

自动提炼规则：
- 主题很短：可直接使用完整主题做主视觉大字
- 主题中等长度：提炼核心词做巨型大字，完整主题进入 B 层
- 主题很长：绝不能整句粗暴放大，必须做设计级提炼

--------------------------------------------------
三、色彩系统：必须更炸、更醒目、更有情境
--------------------------------------------------

整体视觉必须接近以下气质：
- trendy editorial poster
- color-block typography poster
- bold magazine cover aesthetic
- retro-modern travel poster
- conceptual type-led design
- vibrant contrast color system
- cinematic pastel-pop palette
- high-impact graphic poster
- bold and stylish art direction
- image and typography fusion

1）色彩强度要求

画面不能只是“有颜色”，而必须“颜色足够抓人”。
色彩应具备以下特征：
- 更高饱和度
- 更明确对比
- 更强视觉张力
- 更鲜明主次关系
- 更有主题情绪指向

不要温吞，不要平均，不要保守。

2）底色与背景系统必须主动变化

背景绝不能默认是普通白底、浅米底、淡灰底。
背景必须根据主题与情绪主动切换为最合适的系统，例如：

- 高饱和纯色底
- 大面积撞色色块底
- 复古电影感天空色底
- 黄昏粉橙渐变底
- 复古绿墙 / 蓝墙 / 红墙 / 黄墙式场景底
- 强烈冷暖对比底
- 建筑立面式底
- 风景切片式底
- 糖果色城市感底
- 情绪化空间底
- 具有戏剧感的深色底或霓彩底
- 具有场景感的环境底，而不是空白底

底色和背景必须服务于主题表达，增强冲击力，而不是只提供一个安全背景。

3）配色必须根据情境变化，而不是固定公式

如果主题偏轻快 / 俏皮 / 生活方式 / 旅行：
优先使用：
- 珊瑚红
- 番茄红
- 柠檬黄
- 湖蓝
- 天蓝
- 薄荷绿
- 粉色
- 奶油白
- 明亮橙色

如果主题偏理性 / AI / 系统 / 信息 / 方法：
优先使用：
- 电光蓝
- 宝蓝
- 冷白
- 黑
- 酸性黄
- 荧光绿少量点缀
- 高对比灰蓝
- 结构性色块

如果主题偏压迫 / 冲突 / 哲思 / 社会观察：
优先使用：
- 深红
- 黑红
- 焦橙
- 冷灰
- 暗蓝
- 强白
- 高压黑底或深色底
- 极强对比色块

如果主题偏复古 / 电影感 / 生活审美 / 杂志感：
优先使用：
- 奶油粉
- 复古黄
- 鼠尾草绿
- 湖蓝
- 橘红
- 藕粉
- 奶油白
- 柔和但饱和的糖果色块

4）撞色规则

- 每张海报主色通常控制在 2–5 个主色
- 必须有一个绝对主色
- 必须有一个冲突色或强调色
- 可以使用大面积纯色底增强记忆点
- 可以使用明显冷暖冲突增强视觉冲击
- 可以使用色块碰撞、上下撞色、左右撞色、前后撞色
- 允许局部极亮色作钉子色
- 禁止平均上色，禁止发灰发闷，禁止普通浅底保守收尾

--------------------------------------------------
四、视觉冲击力增强规则
--------------------------------------------------

画面必须比普通设计更有第一眼抓力。

请主动增强以下视觉手段中的至少 4–6 种：

- 超大标题压满画面
- 极限放大并裁切文字
- 高饱和背景
- 大面积撞色色块
- 图像与文字强遮挡关系
- 极强前后景层次
- 局部超近景构图
- 非对称构图
- 斜切构图或横冲式文字节奏
- 纵向压缩字形或横向延展字形
- 文字穿出边界
- 图像顶住版面边缘
- 强烈大小比例反差
- 强主视觉 + 强底色 + 小图锚点
- 让观者第一眼就被“颜色 + 文字 + 结构”抓住

画面不能只是“高级”，还必须“有记忆点”。

--------------------------------------------------
五、构图系统
--------------------------------------------------

整个画面必须清晰分为三层：
- 第一层：巨型主视觉文字
- 第二层：主题隐喻图像
- 第三层：完整标题 / 副标题 / 标签 / 编号 / 小字系统

构图要求：
- 巨型文字必须占据主导地位
- 文字可跨页、压边、裁切、压底、竖排、横向延展
- 图像位置必须聪明，像被版式收编，而不是随意贴图
- 留白可以有，但必须是“聪明留白”，不能因为没想法而空
- 背景必须和文字、图像共同构图
- 整体应简洁，但绝不平淡

推荐构图方式：
- 巨型文字覆盖大半画面，图像嵌于中下部
- 大色块背景 + 超大字 + 一个核心图像
- 文字压住场景图像边缘，形成遮挡关系
- 背景分区撞色，文字跨越两个色区
- 文字像建筑外立面，图像成为其中局部空间
- 左右分栏撞色构图
- 上下切割撞色构图
- 图像居中偏下，标题压顶或压底形成海报张力

--------------------------------------------------
六、文字重构系统：必须真正加强
--------------------------------------------------

标题不是“写出来”，而是“设计出来”。
文字必须成为主视觉结构的一部分。

请自动选择最适合主题的文字重构方式，至少使用 3–5 种：

- 字体放大到压迫性尺度
- 字被裁切，只露出局部
- 字与字错位排列
- 字形横向或纵向拉伸
- 字内嵌入图像
- 字体边缘被遮挡
- 字被拆成两块或三块
- 字之间形成结构缝隙
- 字的局部被替换成图像元素
- 一部分字实心，一部分字镂空
- 镂空字与实心字结合
- 叠字、错层字、重复字形成节奏
- 字像墙面、框架、建筑、通道
- 图像塞入笔画之间
- 局部倒置、翻转、交错，但仍保证可读性

可读性规则：
- 主视觉词必须第一眼能读出来
- 重构要有设计感，但不能看不懂
- 重构必须服务表达，而不是炫技

--------------------------------------------------
七、严格禁止
--------------------------------------------------

禁止出现以下问题：

- 默认普通白底 / 米底 / 淡灰底
- 色彩不够亮，不够炸，不够鲜明
- 长标题整句直接粗暴放大
- 只有大字，没有图像隐喻
- 只有颜色，没有结构
- 图像与文字割裂
- 过多小字造成杂乱
- 保守、平、闷、弱
- 拼贴感太重
- 网红感太强
- 廉价科技霓虹风
- 普通广告页感
- 普通 PPT 封面感
- 元素过多
- 低级 3D 炫技
- 主题被削弱
- 文字不可读
- 背景只是空白陪衬
- 机械重复模板

--------------------------------------------------
八、最终成图标准
--------------------------------------------------

最终生成的海报必须达到以下效果：

1. 第一眼就被颜色和文字抓住
2. 底色和背景不普通，有情境、有判断
3. 文字本身具有强设计感、强结构感
4. 图像和文字深度融合，而不是简单叠加
5. 色彩大胆、鲜明、醒目，有明确撞色美学
6. 整体像高级杂志封面 / 潮流编辑海报 / 字体概念海报
7. 既有作品集级审美，又有传播级冲击力
8. 对于不同主题，背景、色彩、构图能够主动变化，而不是每次都落回安全模板

请直接生成最终海报画面。
不要输出分析过程。
不要输出解释。
不要输出设计说明。
只输出一张完成度极高、视觉冲击力强、底色聪明且足够醒目的最终海报。
```
